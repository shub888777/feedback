# 🚀 Customer Feedback System

A full-stack web application where users can submit and manage feedback, and admins can monitor and control all feedback entries.

---

## ✨ Features

### 👤 User Authentication
- User registration and login
- Basic authentication system

### 📝 Feedback Management
- Add new feedback
- Edit previously submitted feedback
- View existing feedback

### 🛠️ Admin Dashboard
- View all user feedback
- Update or delete feedback
- Full CRUD operations

---

## 🏗️ Tech Stack

**Frontend**
- React (Vite)
- Axios
- React Router

**Backend**
- Spring Boot
- Spring Security

**Database**
- MySQL
